An easy-to-use program for analyzing Drosophila Activity.

## Usage

Install the [PyPI package](https://pypi.org/project/easyFlyTracker/):

```commandline
pip install easyFlyTracker
```
or 
```commandline
pip install -i https://pypi.org/simple/ easyFlyTracker
```
